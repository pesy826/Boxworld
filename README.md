# 盒世界 BoxWorld

> 一个微信界面风格的 AI 角色聊天应用。灵感来自 SillyTavern（酒馆），但以「微信 IM」的形态呈现——角色会像真人一样用多条短消息聊天、发朋友圈、主动找你、甚至发表情和图片。

## ✨ 功能特性

- **微信 IM 聊天**：角色以第一人称、多条短消息的方式聊天（非小说式长文），有打字节奏
- **场景模式**：传统 RP 形态（线下见面），支持流式输出
- **虚拟时间系统**：角色有"生命感"——会主动发消息、发朋友圈、评论互动（基于补算心跳调度）
- **朋友圈**：发图 / 点赞 / 评论 / 可见范围（公开 / 单卡世界）
- **群聊**：建群、AI 智能拉群、群内加好友；支持粗略 / 精细两种调度模式
- **NPC 系统**：一句话生成世界里的配角，自动分配头像
- **世界记忆**：统一世界事件记忆 + 角色私有记忆同步
- **单卡模式**：每张卡可拥有独立时间线
- **酒馆生态兼容**：导入 SillyTavern 角色卡（PNG / JSON）、世界书、预设系统
- **素材库**：内置表情包 + 头像库，角色可像真人一样发表情；用户也可发图（vision 直读）
- **ComfyUI 文生图**（仅桌面端）：角色可在聊天 / 朋友圈里发出 AI 生成的照片
- **语音通话**（需中转站支持 STT/TTS）：仿微信全屏语音通话，VAD 自动断句 + 边收边播
- **数据本地化**：纯前端应用，无后端，所有数据存在本地浏览器（IndexedDB）

## 🚀 使用方式

### 方式一：直接下载安装包（推荐普通用户）

前往本仓库的 **[Releases](../../releases)** 页面下载：

- **Windows 桌面版**：`boxworld_x.x.x_x64-setup.exe`（NSIS 安装包）或 `.msi`，双击安装即可，**无需安装浏览器或 Node 环境**。
- **安卓版**：现代手机下载 `app-arm64-release.apk`（绝大多数安卓机适用）；老旧 32 位机下载 `app-arm-release.apk`。

> 桌面版的 ComfyUI 文生图功能仅在桌面端可用；语音通话需要你在设置里填一个支持 `/audio/transcriptions` 和 `/audio/speech` 的中转站。

### 方式二：用浏览器跑（需要 Node.js 环境）

如果你装了 [Node.js](https://nodejs.org/)（建议 18+），可以双击项目根目录的 bat 脚本：

- **`启动-开发模式.bat`**：启动开发服务器（热更新），自动打开浏览器 `http://localhost:5173`
- **`启动-浏览器版.bat`**：先构建生产版本再用本地服务预览，自动打开浏览器

> 浏览器版没有桌面端独有的 ComfyUI 文生图能力（该功能依赖 Tauri）。其余功能均可用。

## 👨‍💻 从源码构建（开发者）

```bash
# 安装依赖
npm install

# 开发模式（浏览器，热更新）
npm run dev

# 构建前端静态产物
npm run build

# 打包 Windows 桌面端（需 Rust 工具链 + Tauri 环境）
npm run tauri build

# 打包安卓（分架构 APK，需 JDK17 + Android SDK/NDK + Rust 安卓目标）
npm run tauri android build -- --apk --split-per-abi
```

### 内置素材包（开发者向）

表情 / 头像素材放在 `public/builtin-assets/` 下，运行脚本生成清单：

```bash
node scripts/build-asset-manifest.mjs
```

详见 `public/builtin-assets/README.md`。
